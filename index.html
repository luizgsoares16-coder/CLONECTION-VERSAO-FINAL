Seja bem-vindo ao Clonection!

Aqui você terá acesso aos especialistas mais qualificados para realizar seu pedido. Está empresa contém um sistema de Match onde você mesmo pode procurar o ator(a) que deseja para a ocasião.

Além de atores profissionalizados você também pode encontrar sosias de artistas famosos que não so aparentam ser iguais mas também possuem vozes idênticas.

O nosso sistema vai funcionar corretamente se você seguir os seguintes passos:

  • Cadastro/Login:

    Caso você ainda não tenha uma conta, para poder utilizar nosso sistema você deve cria-la. Caso já tenha uma conta, basta apenas logar-se nela para poder utilizar o sistema

  • Match:

    O sistema de Match funciona da seguinte forma, em sua tela ira aparecer atores ou sosias que talvez te interessem, você tera duas opções, recusar ou aceitar, se você recusar ira aparecer outra pessoas, caso aceite e o ator(a) aceitar o pedido vocês entrarão em Match e começarão a trabalhar juntos no objetivo pedido.

Caso queira saber mais sobre nós, clique no botão "Descubra o Clonection!".
